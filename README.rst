foxtrot
=========

demo pyxel based space game. not worth a second glance :)

.. image:: https://travis-ci.org/narfman0/foxtrot.png?branch=master
    :target: https://travis-ci.org/narfman0/foxtrot

Getting started
---------------

Install dependencies with::

    make init

Run app with::

    make run

Testing
-------

    make test
